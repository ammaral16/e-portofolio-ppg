# E-Portfolio PPG Ammar Al Hakim Asshiddiqi

Website E-Portfolio untuk mahasiswa Program Profesi Guru (PPG) Informatika Universitas Negeri Yogyakarta.

## Fitur

- **Desain Profesional**: Tampilan modern dengan tema biru navy yang elegan
- **Responsive**: Kompatibel dengan desktop, tablet, dan mobile
- **Navigasi Mudah**: Menu yang terorganisir dengan smooth scrolling
- **Profil Lengkap**: Informasi pribadi dan akademik detail
- **Artefak Pembelajaran**: Kumpulan modul, bahan ajar, media pembelajaran, dan asessmen
- **Refleksi**: Dokumentasi perjalanan selama PPG
- **Kontak**: Form kontak dan media sosial
- **Preview & Download**: Setiap dokumen dapat dipreview dan didownload

## Struktur Menu

1. **Beranda** - Halaman utama dengan pengenalan diri
2. **Profil** - Informasi lengkap mahasiswa
3. **Artefak Pembelajaran**:
   - Modul Pembelajaran (3 modul)
   - Bahan Ajar (3 materi)
   - Media Ajar (3 presentasi)
   - Asessmen (3 instrumen penilaian)
4. **Refleksi** - Pemikiran dan evaluasi selama PPG
5. **Kontak** - Informasi kontak dan form pesan

## Cara Menggunakan

### Local Development
1. Clone atau download repository ini
2. Buka folder project di terminal
3. Jalankan server lokal:
   ```bash
   python -m http.server 8000
   ```
4. Buka browser dan akses `http://localhost:8000`

### GitHub Pages Deployment
Website ini dapat di-deploy menggunakan GitHub Pages:
1. Fork atau clone repository ini
2. Push ke GitHub repository Anda
3. Aktifkan GitHub Pages di repository settings
4. Pilih source: Deploy from branch → main → / (root)
5. Website akan online di: `https://[username].github.io/[repo-name]`

### Google Sites Integration
Website ini dirancang untuk kompatibel dengan Google Sites. Anda dapat:
1. Upload semua file ke Google Drive
2. Set folder sebagai public
3. Embed website ke Google Sites menggunakan iframe
